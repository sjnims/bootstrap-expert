# Components Reference

Quick reference for all Bootstrap 5.3 component classes.

## Accordion

| Class | Description |
|-------|-------------|
| `.accordion` | Wrapper container |
| `.accordion-item` | Individual item |
| `.accordion-header` | Item header |
| `.accordion-button` | Toggle button |
| `.accordion-button.collapsed` | Collapsed state |
| `.accordion-collapse` | Collapsible body wrapper |
| `.accordion-body` | Content area |
| `.accordion-flush` | Remove borders/radius |

## Alerts

| Class | Description |
|-------|-------------|
| `.alert` | Base class |
| `.alert-{color}` | Color variant (primary, secondary, success, danger, warning, info, light, dark) |
| `.alert-dismissible` | Enable dismiss button |
| `.alert-link` | Link styling within alert |
| `.alert-heading` | Heading within alert |
| `.btn-close` | Close button |
| `.fade` | Enable fade animation |
| `.show` | Show state |

## Badges
