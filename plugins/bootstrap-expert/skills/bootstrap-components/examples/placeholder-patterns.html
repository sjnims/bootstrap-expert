Accessibility Notes

Accessibility Requirements
  • Add aria-hidden="true" to placeholder containers to hide from screen readers
  • Provide alternative loading announcements using aria-live regions
  • Replace placeholders with actual content once loaded
  • Consider users who prefer reduced motion (animations respect prefers-reduced-motion)
Screen Reader Example
<!-- Hidden placeholder -->
<div aria-hidden="true">
  <span class="placeholder col-6"></span>
</div>

<!-- Live region for announcements -->
<div aria-live="polite" class="visually-hidden">
  Loading content...
</div>
