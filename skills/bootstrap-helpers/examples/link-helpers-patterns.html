Important Considerations

Tips for Stretched Links:
  • Parent must have position: relative (cards have this by default)
  • Only one .stretched-link per container
  • Other links/buttons inside won't be clickable unless they have position: relative and higher z-index
  • Screen readers only announce the link text, so make it descriptive
