Flush Accordion

Add .accordion-flush to remove borders and rounded corners for edge-to-edge rendering.

The flush variant removes the default background-color, borders, and rounded corners. Useful when nesting inside cards or other containers.

Notice how the items blend seamlessly with the parent container's edges.

This is particularly useful for sidebars, settings panels, or embedded content.
