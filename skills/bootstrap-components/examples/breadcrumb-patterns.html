Accessibility Notes

Accessibility Requirements
  • Always wrap breadcrumbs in a <nav> element with aria-label="breadcrumb"
  • Use <ol> (ordered list) to indicate sequence
  • Add aria-current="page" to the last item to indicate current page
  • Ensure sufficient color contrast for links and text
