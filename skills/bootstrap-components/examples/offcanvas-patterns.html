Bootstrap Offcanvas Patterns

Start (Left) Offcanvas

Default position, slides in from the left.

Left Offcanvas

This offcanvas slides in from the left side of the screen.

Use .offcanvas-start for this position.

End (Right) Offcanvas

Slides in from the right side.

Right Offcanvas

This offcanvas slides in from the right side of the screen.

Use .offcanvas-end for this position.

Top Offcanvas

Slides down from the top.

Top Offcanvas

This offcanvas slides down from the top. Use .offcanvas-top for this position.

Bottom Offcanvas

Slides up from the bottom.

Bottom Offcanvas

This offcanvas slides up from the bottom. Use .offcanvas-bottom for this position.

Static Backdrop

Click outside won't close. Use data-bs-backdrop="static".

Static Backdrop

This offcanvas won't close when clicking outside. You must click the close button.

Options:

  • data-bs-backdrop="static" - No dismiss on click
  • data-bs-backdrop="false" - No backdrop at all
  • data-bs-scroll="true" - Allow body scroll

Responsive Offcanvas

Offcanvas on small screens, visible content on larger screens.

Large screens: The content below is visible inline. Resize to see the offcanvas behavior.
Responsive Offcanvas

This content is:

  • An offcanvas on screens smaller than lg
  • Visible inline on lg screens and up

Use .offcanvas-{sm|md|lg|xl|xxl} for responsive behavior.
