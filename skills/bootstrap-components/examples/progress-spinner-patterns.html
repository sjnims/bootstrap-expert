Accessibility Notes

Accessibility Requirements
  • Progress bars: Include role="progressbar", aria-valuenow, aria-valuemin, aria-valuemax, and aria-label
  • Spinners: Include role="status" and .visually-hidden text describing the loading state
  • Button spinners: Use aria-hidden="true" on spinner if visible text is present
  • Update aria-valuenow dynamically when progress changes
